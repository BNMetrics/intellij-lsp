# intellij-lsp
Plugin adding support for Language Server Protocol for IntelliJ     

What's working :      
Requests to the server:     
-Hover     
-DocumentHighlight     
-Completion     
-didSave, didClose, didOpen, didChange     
     
Client :      
-showMessage     
-logMessage

There is a skeleton of an LSP plugin for Dotty in the intellij-lsp-dotty folder.
